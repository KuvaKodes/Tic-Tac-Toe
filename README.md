# Tic Tac Toe

A simple two-player, same-screen Tic-Tac-Toe game built with React and Vite.

There is no AI opponent — this is human vs. human, played by taking turns on the same device. The game tracks whose turn it is (X or O), detects wins and ties automatically, and includes a reset button to start a new round without reloading the page.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app in development mode:

```bash
npm run dev
```

This starts the standard Vite dev server (by default at `http://localhost:5173`) with hot module reloading.

Other available scripts:

```bash
npm run build    # Build for production
npm run preview  # Preview the production build locally
npm run lint     # Run ESLint
```

## Tech Stack

- React 18
- Vite 6
- ESLint
